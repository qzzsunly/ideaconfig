## Features

* svn commit
* svn diff
* svn revert
* svn update
* svn add
* svn rename
* svn log
* svn blame
* svn lock
* svn unlock

## Requirements

**TortoiseSvn must be installed.**

## Installation
Launch VS Code Quick Open (Ctrl+P), paste the following command, and press enter.
```
ext install tortoise-svn-for-vscode
```
